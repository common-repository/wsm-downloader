== Description ==


**WordPress Social Media Downloader**


This Plugin allows you to download media (photo , video , audio) from instagram , facebook and bandcamp with one-click.


WSMD consists of two parts :

**frontend downloader** : Allow your site visitors to download media from supported social networks for free.


**backend(Admin/Dashboard) downloader** : if you want to add media from Facebook or instagram into your blog posts directly , you can use this feature.this part allows you to download any type of media from anywhere on WordPress Media library just with one click.further more supporting social networks, this section also supports direct download from any media file link.for example if you see a photo in google and you need to use this photo in your post content , you can copy that photo link and paste it in wsmd downloader form and download that in your site directly.

== Installation ==

1.Install the the plugin either via the WordPress dashboard, or by uploading the files to your web server (in the /wp-content/plugins/ path).

2.Activate the th plugin through the ‘Plugins’ menu in WordPress.

3.Navigate to the ‘WSM Downloader’ settings page to configure settings.

4.Use the shortcode [wsmd_downloader] in your page, post or widget to display "Download Form".

5.[Read this Documentation](https://wsmd.ttmga.com/Documentation/)
